# AGENTS.md

## 一、语言偏好设置

请始终使用 **中文（简体）** 进行回复和撰写文档。

### 具体要求：
- 所有对话回复使用中文
- 所有文档、注释、说明使用中文
- 代码中的变量名、函数名、类名等可以保持英文（遵循编程规范）
- **代码注释使用中文**
- 技术术语可在首次出现时提供 **中英文对照**，后续优先使用中文

### 例外情况：
- 代码本身（变量名、函数名、类名等）遵循项目现有规范
- 当用户**明确要求使用英文**时
- 某些 **必须使用英文** 的技术配置文件（如：JSON / YAML / TOML / CI 配置等）

## 二、测试

测试框架使用 Vitest（维特斯特）

### 具体要求：
- 所有测试脚本存放在 src/__tests__/
- 优先编写 API 集成测试，避免使用浏览器端 E2E 测试
